Project Timeline

Our capstone project follows a structured timeline with specific goals and milestones. Below is our weekly breakdown of tasks and deliverables.

Project Progress
40%
Week 1

Project Initialization

  • Initial team meeting and role assignment
  • Brainstorming and concept development
  • Research on existing similar projects
  • Creation of preliminary project outline
Week 2

Technical Planning

  • Finalize project scope and goals
  • Create detailed project requirements
  • Research technical feasibility and limitations
  • Initial prototype sketches and mockups
Week 3

Software Architecture

  • Define complete software architecture
  • Research and finalize libraries for web audio/image processing (OpenCV.js, TensorFlow.js, Tone.js)
  • Write detailed software technical specification
Week 4

Initial Implementation

  • Prototype initial music player interface (basic piano roll and audio playback)
  • Implement basic image processing pipeline (simple shape detection)
  • Establish clear interface/schema between music player and image processing modules
  • Create website to track progress
Week 5

Initial Prototyping

  • Exploration of different wall attachment mechanisms
  • Research on camera and projector options
  • Initial tests with image processing libraries
  • Preliminary sound design explorations
Week 6

Hardware Development

  • Finalize design of interactive wall
  • Test different attachment prototypes
  • Begin construction of camera/projector stand
  • Initial hardware-software integration tests
Week 7

Advanced Implementation

  • Improve shape detection algorithm accuracy
  • Enhance music generation features
  • Develop projection mapping capability
  • Begin user testing with basic prototype
Week 8

Integration & Testing

  • Full system integration
  • Performance optimization
  • Comprehensive testing and bugfixing
  • User experience improvements based on feedback
Week 9

Final Implementation

  • Complete all core functionality
  • Polish user interface and interaction design
  • Prepare demonstration setup
  • Final quality assurance testing
Week 10

Project Completion

  • Final system assembly and configuration
  • Prepare project documentation
  • Create presentation materials
  • Capstone project demonstration
