About Melody Wall

Integrating physical interactions with digital music generation

Project Overview

Melody Wall integrates physical interactions with digital music generation through a Piano Roll interface, leveraging image processing and real-time projection to enable musical expression without the need for musical training or instruments.

Our interactive wall allows users to place physical objects which are detected by our system and transformed into musical notes. The result is an intuitive and engaging way to create music through spatial arrangements.

The project combines elements of computer vision, web technologies, and physical computing to create an accessible platform for musical creativity.

Technical Specification

Software Components

  • Image Processing Pipeline: Webcam input processing
  • Shape Detection: Using TensorFlow.js and OpenCV.js
  • Music Player Software: Piano roll interface
  • Audio Engine: Web Audio API for MIDI sound generation
  • Mapping: Schema to map detected shapes to musical notes

Integration

  • Synchronization between image processing output and piano roll software
  • Projection software displaying processed shapes
  • Real-time piano roll playback visualization
  • Interactive feedback system

Projection & Calibration

  • Real-time calibration of camera/projector setup
  • Accurate interaction mapping
  • Adjustable projection parameters
  • Auto-correction mechanisms

Hardware Components

  • Attachments & Prototypes: Velcro, slat, magnet, pinhole, suction cup
  • Interactive Wall: Custom-designed with attachment mechanism
  • Camera/Projector Stand: Adjustable for accurate calibration
  • Speakers: Audio output system

Academic Affiliation

University of Toronto - Department of Computer Science

This project is developed as part of CSC490: Capstone Design Project at the University of Toronto.

The CSC490 course challenges students to create computer-driven systems that interact with people in the real world through various sensors and actuators, providing novel and compelling experiences that serve a social good.

Melody Wall aims to make music creation more accessible and intuitive through physical interaction, encouraging collaborative creativity and expression without requiring traditional musical training.
